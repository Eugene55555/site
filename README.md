# site — инструменты сборки и сайт KDP

Сюда вынесены инструменты генерации сайта/дашборда из репозитория `kdp-j`,
чтобы `kdp-j` остался чистой библиотекой HTML-исходников книг.

## Состав
- `docs/index.html` — раскрытый сайт (landing/дашборд)
- `build_dashboard.py` — сборка дашборда
- `build_manuscript.py` — сборка манускрипта книги
- `clean.py`, `fix_html.py`, `dbg.py` — вспомогательные скрипты обработки HTML

## Использование
Скрипты работают с HTML-исходниками книг (cover.html / manuscript.html),
которые лежат в репозитории `kdp-j` по пути `<Book Name>/{P,H}/`.
